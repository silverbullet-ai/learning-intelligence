# NumPy

## Overview

NumPy (Numerical Python) is the foundation of scientific computing in Python.

It provides:
- Fast multidimensional arrays
- Mathematical operations
- Statistical functions
- Vectorized computations
- Efficient memory usage

NumPy is heavily used in:
- Data Analysis
- Machine Learning
- Deep Learning
- Scientific Computing

## Installation

pip install numpy

import numpy as np

## Topics Covered

- Array Creation
- Array Attributes
- Reshaping
- Vectorized Operations
- Universal Functions
- Indexing & Slicing
- Statistical Operations
- Boolean Indexing

## Why NumPy Matters

NumPy introduces vectorized computation, allowing operations on entire arrays without explicit loops.
